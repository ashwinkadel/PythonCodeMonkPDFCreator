# PythonCodeMonkPDFCreator
This script create pdf of codemonk tutorial of hackerearth  

### Libraries Require
* bs4
* requests
* fake_useragent
* pdfkit

> This script scrape data from hackerearth codemonk tutorial and convert it into pdf.
